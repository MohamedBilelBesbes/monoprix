Which format sohuld I respect upon submitting the work ?

it is required to submit a dataset by the following format :




You basically need to make sure that you are putting your input in this correct order :



  • The first column is the Date column, the format of the date is as follows: day/month/year(wih four digits)
  • The second column is the shop_id column hich is an integer necessarily
  • The third column is the item_id column; please make sure to not put two different item_id values for the same item across two different data instances
  • The fourth column is the item_category column; please make sure to not put two different item_category values for the same item across two different data instances
  • The fifth column is the id_struct column; please make sure to not put two different id_struct values for the same item across two different data instances
  • The sixth column is the Price column; it sholud be putten as float, not as an integer. Note that it is OK to put different prices for the same product through different dates(prices of a given product does change from time to time)
  • The seventh column is the item_cnt_day; you should put it as integer and it has to be a strictly positive integer


Note that :
Please note that if you get in data of a certain (shop_id,item_id) pair, get all the data specified for that month because even if you submit data of a certain month that is already in the dataset, it will get deleted and replaced by the new data
